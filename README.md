# UnGPT — Unnecessary Garbage Pruner & Transformer

UnGPT is a simple tool that cleans text from unnecessary typography and visual noise.

It removes:

* emojis
* decorative symbols
* excessive formatting

And leaves you with clean, readable text.

---

## Why

Many texts (especially AI-generated ones) are overloaded with:

* emojis
* fancy Unicode symbols
* visual clutter

This makes them harder to read and understand.

UnGPT solves this by stripping everything down to the essentials.

---

## What it does

* removes emojis
* cleans decorative Unicode (arrows, symbols, etc.)
* strips unnecessary formatting characters
* preserves the original meaning
