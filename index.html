Unnecessary Garbage Pruner & Transformer

Turns polished typography into simple, everyday web-style text.

What this does
  • — / – → " - "
  • “ ” „ « » → "
  • ‘ ’ → '
  • … → ...
  • non-breaking spaces → normal spaces
  • collapse runs of spaces
  • fix spacing around punctuation
  • optional: strip markdown (** / *, _italic_, * bullets → -)
  • optional: remove emoji (runs first when enabled, then the rest)
  • optional: trim each line’s edges — last step after typography
